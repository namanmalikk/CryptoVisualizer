<---- This project is made using django and chart.js ---->

1. To check the python code logic go to CryptoVisualizer/views.py 

2. To check the html/css templates and chart.js javascript go to CryptoVisualizer/templates/

3. To run the project download the following dependencies using pip
    pip install django
    pip install pandas
    
4. run the project by running these commands in the folder containing manage.py
    python manage.py migrate
    python manage.py runserver

5. This project uses various cryptocurrency datasets (go to folder named "Datasets/" )

6. Github Link: https://github.com/namanmalikk/CryptoVisualizer
